# ШПАРГАЛКА: МОДУЛЬ 1 (БАЗОВАЯ ЛОГИКА)

## 1. ПРАВИЛА ИСТИННОСТИ (LIST VERSION)
- AND (&&): Истина только когда ВСЕ "да". (1 && 1 = 1)
- OR (||): Истина, когда хотя бы ОДИН "да". (0 || 1 = 1)
- NOT (!): Инверсия. Делает из "да" "нет" и наоборот.
- XOR (разные): Истина только когда значения НЕ ОДИНАКОВЫ.

## 2. ЗАКОНЫ ДЕ МОРГАНА (РАСКРЫТИЕ СКОБОК)
- Отрицание "И": !(A && B) => !A || !B
- Отрицание "ИЛИ": !(A || B) => !A && !B
*Правило: Меняем знак на противоположный, навешиваем "!" на каждый элемент.

## 3. ПРИОРИТЕТЫ (КТО ГЛАВНЕЕ)
1. Скорбки ( )
2. Отрицание !
3. Логическое И &&
4. Логическое ИЛИ ||

## 4. КОРОТКОЕ ЗАМЫКАНИЕ (ИНЖЕНЕРНЫЙ ХАК)
- Объект && Объект.метод() — Запуск только если Объект существует.
- Значение || "Дефолт" — Подстановка, если левая часть пустая.
// Твой вариант (понятный):
if (isReady) {
    sendData();
}

// Инженерный вариант (короткое замыкание):
isReady && sendData();

## 5. ЦИКЛЫ (МЕХАНИКА)
- for (let i = 0; i < 10; i++) — Когда знаем "сколько раз".
- while (условие) — Когда ждем события (крутимся, пока "да").
- do { ... } while (условие) — Сначала делаем, потом проверяем.
