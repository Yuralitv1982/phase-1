## 🔨 2. Дрочка (Drills) (80%)
- [x] Сет задач 2
	Уровень 2: Продвинутый новичок (Type Coercion & Primitives)
	Фокус: Преобразования типов и предсказуемость результата.

		StringToNum: Преврати "42px" в число 42 через parseInt.
	
	    Explicit: Преврати число 0 в булево значение, строку и обратно в число.

	    MathString: Выведи результат: "10" / "2" + 5.

	    NullOps: Выведи результат: null + 1 и undefined + 1. Сравни их.

	    BigInt: Создай два BigInt числа и перемножь их.

	    BooleanTrap: Выведи !! для пустой строки, пробела " ", 0, [] и {}.

	    Concat: Сложи три переменные: строку, число и булево. Посмотри, что получится.

	    StrictCheck: Напиши сравнение null == undefined и null === undefined.

	    NanCheck: Проверь через isNaN, является ли результат "abc" * 10 числом.

	    PromptSum: Через prompt возьми два числа и выведи их сумму (не склейку!).
